# Campaigner Enterprise

A specialized AI-powered platform for scaling Direct Marketing campaigns (Email, SMS, WhatsApp, Push) with quality assurance and governance.

## Features
*   **AI Copywriter**: Generates subject lines and body copy based on brand tone.
*   **Brand Guardian**: Validates content against forbidden words and style guides.
*   **Legal Compliance**: Checks for required disclaimers and valid dates.
*   **Workflow Automation**: Full lifecycle from Draft -> Optimization -> Approval.

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/CristhianForero08/campaigner-app.git
    cd campaigner-app
    ```

2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3.  Run the application:
    ```bash
    python main.py
    ```

4.  Open the UI:
    *   Go to http://localhost:8000
